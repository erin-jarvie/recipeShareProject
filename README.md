A full-stack app with:
- Backend (.NET)
- Frontend (Angular)
- Tests (xUnit)

Required
- .NET 9 SDK
- Entity Framework
- SQLite
- Angular
- fontawsome
- bootstrap
  

- Frontend: Angular -> Backend: RecipeManager (api) -> RecipeMangerController -> DataAccess: AppDbContext -> Datbase: SqlLite

- Latency over 500 requests = 1.71 ms (Benchmark dot net)
